# dv

![](static/images/iconf.jpeg)

给自己用的 alfred workflow.

# usage 

require :

* go1.16+
* alfred 4.5+

![](static/images/dvv.gif)

# install 

1. clone repo 
2. cd 至 repo 根目录，修改 configs/config.json
3. cd 至 repo 根目录，`make` dv binary
4. 导入 repo 根目录下 `dv.alfredworkflow` 至 alfred
